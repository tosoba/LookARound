## Getting Started


1. Go to [https://developers.nextzen.org/](https://developers.nextzen.org/) and sign up for an API KEY.
2. Paste the API key into `gradle.properties`
   ```properties
   nextzen.apiKey=<NEXTZEN_API_KEY>
   ```
